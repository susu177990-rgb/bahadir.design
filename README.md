# 巴哈地尔 · 作品集

个人作品集网站，技术向、深色主题。

## 技术栈

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript

## 本地开发

```bash
npm install
npm run dev
```

访问 http://localhost:3000

## 构建与部署

```bash
npm run build
```

部署到 Vercel 并绑定域名 bahadir.design。

## 内容更新

- **项目**：编辑 `data/projects.ts`
- **关于页**：编辑 `app/about/page.tsx`
- **联系页**：编辑 `app/contact/page.tsx`
- **简历**：将 PDF 放入 `public/resume.pdf`
- **作品图**：替换 `projects.ts` 中的 `thumbnail` 为真实图片路径
